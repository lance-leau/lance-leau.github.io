Project Presentation

Controll Settings

  • Delete - deletes the given shape if at least one other shape is left
  • Size - modifies the size of the shape
  • FOV - modifies the FOV of the camera
  • Center - shows the center of rotation with a red dot
  • Vertices - shows the 27 vertices that forms the edges can connect to
  • Sensitivity - controls how fast the shape rotates when dragged by the mouse

Make your own shape !

  • Point 1/2: control wich point to move
  • Arrows - move point to next vertex
  • Add Edge - Add an edge (a line) to the shape
  • Undo - removes the last added edge, can be repeted
  • Save - add shape to saved list, you can now create a new shape
  • Delete - deletes the shape
